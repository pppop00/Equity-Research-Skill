阳狮集团在最新财年材料中披露 FY2025 集团净收入约 €14,547m、有机增速约 +5.6%；其中 Connected Media 约占净收入 约 60%（约 €8,728m 量级）。该分部是 Epsilon 能力栈所处的「收入池」，但并不等同于 Epsilon 单体收入——其中亦包含媒介购买、代理服务与其他高相关收入。2019 年集团对 Epsilon 体系的大额收购奠定了「第一方数据 + 忠诚度 + 零售媒介 + CDP/身份」叙事；历史文件中曾出现收购前 约 $1.9bn 净收入（2018 口径）等背景数字，仅宜作沿革参考。

业务层面，Epsilon 的核心是 CORE ID / PeopleCloud 生态内的身份解析与受众构建，以及 忠诚度计划、零售媒介与营销自动化 的打包交付；与 Microsoft、LiveRamp 等伙伴的联盟有助于在企业级客户侧降低集成摩擦。上行驱动力来自零售数据网络扩张与可测量闭环预算；下行压力来自 全球隐私合规成本、Cookie 受限带来的测效缺口，以及大型科技与零售商 walled garden 对独立数据中介的利润挤压。

核心亮点

  • 集团战略层面「数据 + 媒介 + 忠诚度」一体化交付的关键拼图，擅长第一方数据与会员运营产品化。
  • 云与身份网络联盟（如 Microsoft / LiveRamp）增强企业级 CDP/分析渗透与交付速度。

  • AI 辅助受众细分、创意与报告自动化可能提高人均产出与客户迭代频率（亦引入治理与品牌安全要求）。

主要风险

  • 监管与同意机制碎片化推高合规成本；执法与集体诉讼尾部不可忽视。

  • 2019 年大额交易在集团层面形成商誉与无形资产，需关注减值测试节奏与叙事扰动。
